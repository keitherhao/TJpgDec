# TJpgDec
http://elm-chan.org/fsw/tjpgd/00index.html

# demo run 方法
    git clone git@github.com:keitherhao/TJpgDec.git
    cd TJpgDec
    mkdir build
    cd build
    cmake ..
    make
    cd ../bin
    ./jpeg_demo

# demo 说明
    支持输入文件和buff地址，只需要写好对应的 input 函数
    输出 只需要 修改 output 函数的实现即可实现 BGRA8888 格式
